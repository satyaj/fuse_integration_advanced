[#ws-security]
== WS Security

* WS Security allows to secure the SOAP Message like :
** Pass Username token with digested, base64 password
** Encrypt messages or parts of messages
** Sign messages
** Timestamp messages

* Implemented mainly using Apache WSS4J with interceptor (IN/OUT)
