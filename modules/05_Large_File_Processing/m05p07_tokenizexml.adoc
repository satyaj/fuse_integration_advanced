[#tokenizexml]
== Tokenize XML

* Goal: Split XML content
* Available using TokenizeXML & XTokernize languages
+
.Example
[source,xml]
----
<from uri="file:inbox"/>
  <split streaming="true">
    <tokenize token="order" xml="true"/>
    <to uri="activemq:queue:order"/>
  </split>
----
